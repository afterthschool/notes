## How to read the notes

- Each subject folder has a `README.md` listing its chapters.
- Each chapter is a folder (`chapter-1/`, `chapter-2/`, …) containing:
  - `index.md` — chapter overview with activities and key concepts
  - `section1.md`, `section2.md`, … — detailed section notes
- Figures live under [`../assets/class-10-ap/`](../assets/class-10-ap/).
